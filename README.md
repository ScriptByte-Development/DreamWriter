# DreamWriter
C# portable code editor
Features:
* Syntax highlighting for many languages
* Automatic indent and outdent
* Handles huge documents
* Highlight matching parentheses
* Displays hidden characters
* Line wrapping
* Code folding
* Live syntax checker
* Plugins/Extensions

Features we want to add:
* * CMD console
* Drag and drop text
